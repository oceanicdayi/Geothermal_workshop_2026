## 💡 提示詞優化建議 (How to Prompt Better)

根據本專案的執行經驗，若希望 AI 產出更精準、更具專業深度的內容，建議採用 **R.T.F.C. 架構** 來優化提示詞：

### 1. 指定角色 (Role)
*   **建議：** 明確告訴 AI「你是誰」。
*   **範例：** `請扮演具備 10 年經驗的「地熱能源產業分析師」或「大學職涯發展顧問」。`
*   **效果：** AI 會自動調整語氣（Tone），從平鋪直敘轉為專業分析或循循善誘。

### 2. 明確任務 (Task) 與痛點挖掘
*   **建議：** 不要只要求「摘要」，嘗試要求「分析痛點」或「對比差異」。
*   **範例：**
    *   ❌ `摘要這場演講。`
    *   ✅ `請整理出講者在實務開發中遇到的 3 個主要痛點 (Pain Points)，以及他們如何克服。`
*   **效果：** 能挖掘出比表面資訊更有價值的深層內容（Insight）。

### 3. 具體格式 (Format) 與細節
*   **建議：** 對於推薦內容，要求「具體名稱」。
*   **範例：**
    *   ❌ `建議學生多學電腦。`
    *   ✅ `請列出學生必學的具體技能樹，包含軟體名稱 (如 QGIS, Petrel) 與程式庫 (如 Python Pandas, ObsPy)。`
*   **效果：** 產出的資訊具備可執行性（Actionable），不再是空泛的建議。

### 4. 上下文 (Context)
*   **建議：** 若是製作網頁，提供風格指引。
*   **範例：** `網頁設計請採用現代化風格，配色需符合「地熱」意象（橘色、深灰色），並確保手機版面閱讀舒適（RWD）。`
